Introdução

O DOM é uma API (Application Programming Interface), Interface de Programação para Aplicativos, destinada a documentos escritos com marcação HTML ou XML.

Definição

O DOM - Document Object Model do W3C é uma interface independente de plataforma e linguagem que permite os programas e scripts acessar e atualizar dinamicamente a estrutura, o conteúdo e a estilização de documentos.

Tipos de DOM

  • DOM para HTML
  • DOM para XML
